# 常用语法



## 1.标题格式

`# 标题内容`为标题格式，注意空格的输入！

```markdown
# 1.标题内容 --一级标题
## 2.标题内容 --二级标题
...
##### 5.标题内容 --五级标题
####### 6.标题内容 --六级标题
```



## 2.代码

代码格式为：

```markdown
`Code`
```

例：`Code`



代码块格式为：

```markdown
​```这里输入语言种类，如：html、markdown等
这里输入你的代码
​```
```

例：

```html
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<meta name="viewport" content="width=device-width, initial-scale=1.0">
	<title>Document</title>
</head>
<body>
	
</body>
</html>
```
